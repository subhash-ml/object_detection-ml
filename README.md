# Object Detection System (Mobile, Charger, Pen)

##  Project Overview
In this project, I developed an object detection system that identifies common items like mobile phones, chargers, and pens. The model analyzes input images and predicts the object along with a confidence score. If the object does not match any of the trained categories, it is labeled as "Unknown".

##  Technologies Used
- Python
- OpenCV
- NumPy

##  How It Works
- The system processes input images
- Detects objects using trained logic
- Assigns a label (Mobile, Charger, Pen)
- Displays confidence score for prediction
- Labels unknown objects when no match is found

##  Features
- Detects multiple objects
- Confidence score for each prediction
- Handles unknown objects effectively
- Simple and efficient implementation
